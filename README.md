# Football Auto Bot ⚽
A Telegram bot that sends live football match summaries every 7 minutes  
for top 5 leagues + UEFA Champions League.

### Railway Setup
Add these variables in **Variables → New Variable**:

| Variable | Example Value |
|-----------|----------------|
| BOT_TOKEN | 8336...SEI8as |
| OWNER_CHAT_ID | 7742985526 |
| API_KEY | your api-football key |
| BOT_NAME | Football Auto Bot |

Then set **Start Command:**  
```
python3 main.py
```

Deploy → Done ✅
